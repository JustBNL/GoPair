# GoPair Admin Frontend

管理后台前端，基于 Vue 3 + TypeScript + Vite 构建。

## 技术栈

- Vue 3 · TypeScript
- Vite 7 · pnpm 9+
- Ant Design Vue 4 · Pinia 3 · Vue Router 4

## 快速启动

```bash
cd apps/backend/gopair-admin-service/admin-frontend
pnpm install
pnpm dev
```

访问 `http://localhost:5174`，后端直连 `http://localhost:8088`（不经过网关）。
