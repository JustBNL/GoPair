# GoPair Frontend

用户端前端，基于 Vue 3 + TypeScript + Vite 构建。

## 技术栈

- Vue 3 · TypeScript
- Vite 7 · pnpm 9+
- Ant Design Vue 4 · Pinia 3 · Vue Router 4

## 快速启动

```bash
cd apps/frontend
pnpm install
pnpm dev
```

访问 `http://localhost:5173`，前端通过 `VITE_API_BASE` 环境变量连接后端网关（默认 `http://localhost:8081`）。
